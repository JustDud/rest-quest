## Troubleshooting

**Port already in use?**
- Change port in `vite.config.js`

**Dependencies issue?**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Build fails?**
- Ensure Node.js 18+ is installed
- Check `node --version`
